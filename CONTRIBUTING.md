# Contributing

Thank you for your interest in contributing to the Kasa Smart Dim integration! We welcome contributions from the community.

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/kasa_smart_dim.git`
3. Install development dependencies: `pip install -r requirements-dev.txt`
4. Set up pre-commit hooks: `pre-commit install`

## Code Style

This project uses [Ruff](https://github.com/astral-sh/ruff) for linting and formatting. Ensure your code passes all checks:

```bash
ruff check .
ruff format --check .
```

## Testing

Run the test suite:

```bash
pytest
```

## Pull Request Process

1. Create a feature branch from `main`
2. Make your changes
3. Ensure all tests pass and code is properly formatted
4. Update documentation if necessary
5. Submit a pull request with a clear description of the changes

## Issues

Report bugs or request features using the [GitHub Issues](https://github.com/yourusername/kasa_smart_dim/issues) page.
